# Sakspilot: Status pr 28. mai 2026

Snapshot av faktisk tilstand i prod. Oppdateres ved hver større leveranse.

## TL;DR

Sakspilot er **funksjonelt komplett for pilotbruk**. Web + API + desktop-agent
er deployet, betalingsforberedende stub er på plass, og produktet er gratis
ut hele 2026 (pilotperiode). Mangler kun: Stripe-aktivering (Q1 2027), Bedrock-
dependency-install (kun ved behov), og partner-godkjenning hos Tripletex (Fiken
er nå løst via personal-access-token).

## Funksjonsdekning

### ✅ Komplett

- **Auth + multi-tenancy**, JWT med tokenVersion-revokering (30s cache), per-organisasjon-isolering, register/login/glemt-passord/bytt-passord/logg-ut-overalt
- **Sak-CRM med kanban**, drag-and-drop status, milepæler, frister, klient-kobling, hourlyRate per sak
- **Klient-håndtering**, kontaktinfo, gjenbruk på flere saker
- **Kalender + Gantt**, frister + milepæler visualisert
- **Klistrelapper**, per sak eller frittstående, fargekoder
- **Agenter / automatiseringer**, Monday/Notion-stil triggere (sak-ferdig → klistrelapp, ny-sak → epost-utkast osv)
- **Tidsregistrering**, manuell + automatisk via desktop-agent (matching-regler basert på vindustittel/applikasjon)
- **Desktop-agent**, tray-app, BrowserView for embedded snarveier, dashboard-vindu
- **Outlook-integrasjon**, Microsoft Graph, automatisk e-post→sak-kobling
- **AI-assistent**, Claude Sonnet 4.5, prompt caching, PII-minimisert (kun klient-navn, ikke epost/telefon)
- **Faktura-eksport CSV**, fungerer mot Tripletex + Fiken-import
- **Fiken direkte**, PAT-basert, oppretter fakturadraft fra billable timer (NY)
- **GDPR**, innsyn, sletting, audit-log, DPA-mal, personvernerklæring
- **Onboarding-veileder**, 9 bransje-valg → bransjespesifikke launcher-snarveier
- **Tema-velger**, 3 fargesett (Navy/Rosa/Skog), `/innstillinger/utseende`
- **Sidebar Mine sites**, PWA-stil ikon-grid for egne live-prosjekter
- **Hjem-tilpasning**, skjul/vis widgets, "Start arbeidsøkt"-CTA
- **Pilot-pricing**, alle org får `pilotUntil = 2026-12-31`, 14-dagers trial etterpå
- **Site-verifisering**, Google Search Console + Bing Webmaster Tools live

### 🟡 Stub / klart for aktivering

- **AWS Bedrock-provider**, abstraksjonen er på plass (`lib/aiProvider.ts`), bare dependency + creds gjenstår. Aktiveres når kunde krever EU-data-residency.
- **Stripe**, `Subscription`-modell + `/billing/status` finnes, men checkout aktiveres først Q1 2027

### 🔴 Avhengig av eksternt

- **Tripletex direkte**, krever partner-status (søknad inne, 3-5 dagers godkjenning)
- **Anthropic ZDR**, krever Enterprise-avtale ($$$). Kompensert med PII-minimisering i prompts

## Datamodell (hovedtabeller)

`Organization` → `User` (1:N, owner/member/admin)
`Organization` → `Client` (1:N)
`Organization` → `Sak` (1:N) → `Milestone`, `TimeEntry`, `MatchingRule`, `EmailLink`
`Organization` → `Automation`, `StickyNote`, `AuditLog`, `Invoice`
`Organization` → `Subscription` (1:1), `FikenIntegration` (1:1, NY)
`User` → `AgentSession` (1:N, én per desktop-installasjon)

Cascade-isolering: hele tre-et henger på Organization, sletting fjerner alt.
