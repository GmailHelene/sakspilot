## Sammendrag

Tre store ting har skjedd siden 3. juni:

1. **Hele tilbakemeldings-rapporten fra ekstern review** (12 områder) er fikset eller dokumentert (PII-stripping, hero-cleanup, CSP-rydding, Inter self-host, sammenligningstabell utvidet, placeholder-sitater skjult, UI-figurer som mockups, responsiv CSS, edit/lokale-filer på snarveier).

3. **CI har nå alle 4 jobs grønne** med dedikert Neon-test-branch. 84 tester kjøres ved hver push, hvorav penger + krypto er dekket (mva, invoiceMath, invoiceLineItems, crypto). Race condition mellom testfiler løst med `fileParallelism=false`.

Resultat: Sakspilot er nå "avansert pilot/MVP" med ærlig dokumentert modenhets-status (ikke "konkurrent på beste SaaS"-formuleringen som ble fjernet etter review). Bredt funksjonelt, sikkerhetsfundamentet er solid, men noen kjente gap (PDF-DoS-isolasjon, paginering, staging-miljø) gjenstår.

### Tester (regnskaps-kritisk + krypto dekket)

## Kjente gap (åpne, dokumenterte)

Detaljer i `SECURITY-NOTES.md`. Kort:

| Risiko | Hva | Plan |
|---|---|---|
| 🔴 | `demo-advokat.mp4` er 67 MB i `/public` — dreper LCP og Vercel-bandwidth | Komprimer til 5-10 MB eller flytt til Cloudflare Stream. Egen runde. |
| 🟡 | 65 sider med `inline-style={{}}` — tema-bytte upålitelig, ingen CSS-extraction | Migrer ved nye sider, ikke en stor sprint |
| 🟡 | Ingen staging-miljø — alt rett til prod | Sett opp før første betalende kunde |
| 🟡 | CSP har `unsafe-inline` og `unsafe-eval` for script-src (kreves av Next.js hydration) | Vurder nonce-basert CSP etter første betalende |
| 🟡 | Render Free-tier (kaldstart 30-60 sek) | Oppgrader til Starter ($7/mnd) før betalende |
| 🟡 | Ingen UptimeRobot-overvåkning | Sett opp 5-min ping på `sakspilot.no` + `api.sakspilot.no` |
| 🟡 | Paginering mangler — frontend henter alle lister | Implementer ved første kunde med 500+ poster |
| 🟡 | Ingen Apple Developer Cert (~$99/år) — Mac-brukere må høyreklikk → Åpne | Når Mac-bruker-base når kritisk masse |
| 🟢 | Backend aksepterer fortsatt Bearer-header (for Electron-desktop) | Kan fjernes når desktop migreres til cookie eller egen API-key |
| 🟢 | Inline-CSS-migrasjon | Pågående, ikke kritisk |
